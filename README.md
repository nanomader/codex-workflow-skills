# Codex Workflow Skills

Reusable Codex skills for repository workflows: prompt rewriting, bug triage,
decision scouting, local review, implementation planning, and PR review polling.

## Skills

- `anti-hivemind-prompt-rewriter`
- `bug-triage-analysis`
- `decision-gap-scout`
- `local-codex-review`
- `plan-author-review-loop`
- `pr-codex-review-poll`

## Notes

These skills are intended to be repository-agnostic. They should not depend on
private local paths, private repositories, or project-specific policy documents.
